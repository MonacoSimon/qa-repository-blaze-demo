## API Testing - BlazeDemo

Se validan endpoints HTML y formularios:

- GET / → disponibilidad
- GET /reserve.php → carga de reservas
- POST /purchase.php → compra
- GET /get-login -> login
- GET /request-reserva -> formulario-vuelo
- GET /index -> pagina principal
- GET /password-reset -> prueba pagina de reseteo de contraseña
- GET /vacation.html -> pagina destino de la semana

### Validaciones

- Status codes
- Content-Type (HTML)
- Tiempo de respuesta
- Casos negativos

### Ejecución

Importar colección y environment en Postman
